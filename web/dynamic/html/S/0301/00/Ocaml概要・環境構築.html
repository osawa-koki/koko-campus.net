Ocaml

Ocamlは1996年にINRIA(フランス国立情報学自動制御研究所)が開発した関数型プログラミング言語で、数的処理を得意とします。
強い静的型付けを行うコンパイラにより、型安全という性質によりバグの少ない高信頼なプログラムが開発できます。

また、関数型言語であると同時にオブジェクト指向もサポートしており、「関数型 × オブジェクト指向」なプログラム開発が可能です。
ガベージコレクション機能がコンパイラに搭載されているため、C言語によるmalloc/freeのようなメモリ管理をプログラマが行う必要はありません。
関数型言語にしては珍しく、GUIアプリケーションの開発も容易に可能です。

関数型言語

プログラムの処理を関数の定義と適用によって行う言語を言います。
関数型言語では関数を値として扱うことができます。
言い換えれば、関数を変数に代入することや関数の引数に関数を渡すこと、さらには関数の戻り値に関数を返すことが可能です。

これに対して、python・C#・Java・C++などは命令型言語と呼ばれます。
これは、これらの言語での文一つ一つが命令であり、これを逐次に実行していくからです。
関数はあくまでもこの処理をサポートする立場にあり、主となる処理は変数に対する代入(更新)であって、変数に関数を適用するという概念はありません。

もっとも、関数型言語と命令型言語は完全に区別されるものではなく、関数型言語に近い命令型言語や命令型言語に近い関数型言語も存在します。
例えば、JavaScriptやC#は命令型言語でありながら、関数を値として変数に代入する評価式型関数の生成が可能です。

opam

opamとはOcaml用のパッケージ管理ツールです。
Ocamlコードをコンパイルしてプログラムを生成するには、このopamを使用することが一般的です。

ということで早速opamをインストールしてみましょう♪ opam 原則としてUNIXライクなOSでのみインストール可能です。
Windows上でOcaml(opam)を使用する場合にはWSLを使用します。
以下のリンクを参考にWSLをインストールしてください。 WSL UNIXライクな環境の構築が完了したら、以下のコマンドを実行します。 bash -c "sh <(curl -fsSL https://raw.githubusercontent.com/ocaml/opam/master/shell/install.sh)"

VScode

エディタは「VScode」をオススメします。
使いやすくて機能が豊富という特徴を持ち、WindowsユーザならばWSLウィンドウを使用してLinux環境で簡単に開発が可能だからです。 VS code 特に設定することなく使用可能です。
左側の拡張機能メニューからOcaml用のプラグインを使用することをオススメします。
WindowsユーザでWSL上にOcaml開発環境を構築した方は、WSLウィンドウを使用することで簡単に操作ができます。

左下のリモートホストボタンから「新しいWSLウィンドウ」を選択します。
WSLに接続したVScodeインスタンスが新たに生成されます。
