dockerコマンド

Dockerでコンテナを操作する際にはdockerコマンドを使用します。
dockerコマンドは以下のように書きます。 docker ～～～ 例えば、Dockerで操作させているプロセス一覧を表示するには以下のコマンドを実行します。 docker ps dockerコマンドには以下のものがあります。
container コンテナ操作系
image イメージ操作系
volume ボリューム操作系
network ネットワーク操作系

Dockerコマンドの構成

Dockerは階層構造をとっており、例えば上で説明した「container」コマンドは配下に以下のコマンドを受け付けます。
  • start
  • stop
  • create
  • run
  • ...

画像は仕組みと使い方がわかる Docker＆Kubernetesのきほんのきほんより。

コマンドとオプション

コマンド(広義)は大きく、コマンド部(狭義)とオプション部に分割できます。
コマンドはその名の通りプログラムに対する命令で、オプションはプログラム実行に際に必要となる補足情報です。
