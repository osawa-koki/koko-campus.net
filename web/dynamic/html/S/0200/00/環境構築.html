C言語

プログラミング言語の基本中の基本の言語です。

1972年にAT&T社のベル研究所で開発されて以来、システムやプログラムの基盤となる部分に用いられている言語です。

「Windows」「UNIX」「Linux」などの主要なOSなどや組込み系OSなどで用いられています。
OSだけでなくアプリケーションの多くがC言語ないしはC言語の派生形であるC++言語で書かれています。

特徴としては習得に時間がかかりますが、高い汎用性と実行効率を持つことが挙げられます。

コンパイルとビルド

コンピュータは我々が書いた英語みたいなコードを直接実行することはできません。
コンピュータが実行することができるのは「0」と「1」だけで書かれたバイナリデータと呼ばれるものだけです。
そのため、我々が書いたコード(ソースコード)をコンピュータが実行できるように変換する必要があります。
これを「コンパイル」ないしは「ビルド」と呼びます。

コンパイル

プログラミングでは、英語っぽいコード(ソースコード)を入力しますがコンピュータはこのコードを直接実行することはできません。

コンピュータが我々が書いたソースコードを実行するためには一度ソースコードをコンピュータが実行できるコード(実行可能プログラム)へ変換する必要があります。

この作業を「コンパイル」と呼びます。

ビルド

コンパイルの進化バージョンです。
入力されたコードをチェックして、さらには実行時に必要なライブラリを結合させて実行可能なプログラムを作成することを言います。
ここでは、コンパイルとビルドを厳密に区別することはしませんが、これ以降にビルドと言ったら「コンパイル + α」の作業だと捉えてください。

コンパイラの取得

C言語でプログラミングをする際にはまずC言語のコンパイラ(コンパイルをしてくれるプログラム)を取得する必要があります。

C言語コンパイラの入手方法を以下の記載します。
  • MinGW
  • clang
  • gcc

といっても、C言語のコンパイラを取得してそれをシェルでコマンドを打って、コンパイルetc...は大変ですので、ここではマイクロソフトが提供しているIDE(統合開発環境)であり、C言語等のコンパイラがプラグインで使用可能で簡単に実行可能な「Visual Studio」の使用を推奨します。

Visual Studioの基本操作

ダウンロード

まずは以下のリンクからからVisual Stadioをダウンロードしましょう♪ VisualStudio 個人でしたら、コミュニティを選択してください。
無料で使用可能です。
ダウンロードが完了したらそのまま起動します。

インストール

ツール(プラグイン)の追加の画面では赤い丸で囲んだ「C++によるデスクトップ開発」をクリックします。

僕はそれ以外もチェックしていますが、「C言語」と「C++言語」のプログラミング環境でしたら、「C++によるデスクトップ開発」だけでok!です。

ツール(プラグイン)のインストールが完了したら最初の画面が現れると思いますが、これは無視してok!です。

プロジェクトの作成

Visual Studioではプロジェクトと呼ばれる単位でファイルを管理します。
まずはプロジェクトを作成しましょう♪

左上の「ファイル」メニューから、新規作成(N)を押すと右側にコンテキストメニューが表示されます。
その一番上のプロジェクト(P)をクリックします。 作成するプロジェクトに関する情報を入力する画面が現れます。
ここでは、C言語を採用するため、言語の部分を「C++」に設定します。

幾つかの候補が現れるため、「空のプロジェクト」ないしは「コンソールアプリ」を選択します。 プロジェクト名は任意に設定してください。

ここでは、デフォルトで表示される「Project1」とします。

項目の追加

次にプロジェクトを構成するファイルを作成しましょう♪
右側のソリューションエクスプローラーの画面を右クリックします。
コンテキストメニューの中央に表示される「追加(D)」をクリックするとさらに詳細なコンテキストメニューが表示されます。
この一番上の「項目の追加(W)」をクリックします。 作成する項目(ファイル)の種類を設定する画面が表示されます。
「C++ファイル(.cpp)」をクリックして、ファイル名を任意に設定します。
ここでは「test」とします。
ここでファイルの拡張子を「.cpp」から「.c」に変更してください。

「.c」(C言語)は「.cpp」(C++言語)でも書くことができますが、一応慣例としてC言語のファイルの拡張子は「.c」と決まっているので、ここでも「.c」で統一します。

入力されたテキストを表示する画面が表示されましたね♪
これで準備完了です!!

Visual Studioの設定

ここでは以下の設定をします。
  • フォントの変更
  • 行間隔の変更
  • 空白文字の表示

フォントの変更

「ツール > オプション > 環境 > フォントおよび色」からフォントを変更します。
オススメは「consolas」または「monospace」です。
また、この画面でフォントサイズも変更できます。

行間隔の変更

「ツール > オプション > テキストエディター > 全般」から行間隔を変更します。
オススメは「1.15」です。

空白文字の表示

行間隔の変更画面と同じ、「ツール > オプション > テキストエディター > 全般」から変更します。
空白の表示にチェックを入れてください。

プログラムの実行

実際になんかの文字を表示してみましょう♪
試しに以下のコードをコピペしてください。
中身は気にしなくてok!です。 #include <stdio.h> int main(void) { printf("%s", "hello world!"); return 0; } メニューのデバッグから「ローカル Windows デバッガー」ないしは「デバッグなしで開始」を選択します。
黒いコンソール画面が現れて「Hello World!...」と表示されたら成功です。
