リスト

リストとは同一の型のデータを複数格納するデータ型です。

異なるデータ型の要素を持つこともできますが、、、

リスト内の要素は「0」から始まるインデックス番号で指定されます。

リストの作成

リストは以下のように作成します。 変数 = ["要素1", "要素2", "要素3"]

リストの要素へのアクセス・リストの走査

リスト内の要素にアクセスするには「リスト名[インデックス番号]」で指定します。
インデックス番号は「0」から始まることに注意してください。 ary = ["ピチュー", "ピカチュウ", "ライチュウ"] print(ary[0]) print(ary[1]) print(ary[2]) リストは全ての要素をfor文を用いて取り出すこともできます。 ary = ["ピチュー", "ピカチュウ", "ライチュウ"] for k in ary: print(k)

リストの追加

リストに要素を追加する方法として以下の2つを覚えましょう♪
  • insert
  • append

insert

pythonでリストの任意の位置に要素を追加するには「insert」メソッドを使用します。
メソッドとはとあるオブジェクト専用の関数ですが、とりあえず「対象オブジェクト.関数名(引数)」で実行すると覚えてください。
対象オブジェクトとはここではリストのことです。
insertメソッドは以下のように実行します。 対象のリスト.insert(追加位置, 追加する要素) 追加位置よりも後ろにある要素はひとつ分後ろに押し出されます。 ary = ["python", "c", "java", "ruby", "php"] ary.insert(0, "fortran") # 先頭に追加 print(ary) ary = ["python", "c", "java", "ruby", "php"] ary.insert(2, "c++") # 先頭から「3」番目に追加 print(ary) マイナス値を追加位置に指定することで後ろからの位置を指定することもできます。 ary = ["python", "c", "java", "ruby", "php"] ary.insert(-2, "c++") # 後ろから「2」番目に追加 print(ary)

append

リストの一番最後に要素を追加するためにメソッドです。
以下のように実行します。 ary = ["python", "c", "java", "ruby", "php"] ary.append("go") # リストの最後に追加 print(ary)

index

あるデータに関してリスト内を探索して一致する要素のインデックス番号を返します。
indexメソッドは以下のように記述します。
一致するデータが見つからなかった場合はエラーとなります。 ary = ["python", "c", "c++", "java", "ruby", "php"] idx = ary.index("c++") print(idx) # 「2」と出力 腕試しとしてindexメソッドを使用しないで自分で関数を作ってみましょう♪
forループでリスト内の要素を走査して一致した際のインデックス番号を返せばok!です。
僕はこう作りました。 def find_it(ary, target): i = 0 for k in ary : if k == target : # マッチしたら、、 return i # そのインデックス番号を返して関数から抜け出す i = i + 1 return -1 # 見つからなかったら「-1」を返却 fint_it関数の第一引数に対象のリスト、第二引数に探したいデータを与えると、戻り値に一致するデータのインデックス番号を返し、一致するデータがなかった場合は「-1」を返します。

要素の削除

リストの要素を削除するには以下のように記述します。 del リスト名[インデックス番号] インデックス番号でしか削除できないので、先ほどのindexメソッドと併せて使用することが多いです。 ary = ["python", "c", "c++", "java", "ruby", "php"] idx = ary.index("java") # javaのインデックス番号を取得 del ary[idx] # javaを削除

二次元リスト

リストの中にリストを入れることもできます。
リストの中にリストを入れた場合二次元リストとなり、いわゆるエクセルのような表のデータを管理することができます。
リスト内の要素を指定するには「リスト名[外側のインデックス番号][内側のインデックス番号]」と指定します。 ここでは簡単に掛け算九九の結果を二次元リストに保存してそれを表示してみましょう♪ kuku = [] # 空のリストを作成(要素を追加するため) for i in range(1, 9 + 1): # 外側のループ kuku2 = [] # 空のリストを作成(要素を追加するため) for j in range(1, 9 + 1): # 内側のループ kuku2.append(i * j) # 九九の結果を内側のリストに追加 kuku.append(kuku2) # 内側のリストを外側のリストに追加 for k in kuku: for l in k : print(str(l) + " ", end="") print("")
