FastAPI

DjangoやFlaskと比べると認知度は低いですが、APIなど提供などの軽量で高速な処理を得意とするPythonで書かれたWEBアプリケーションフレームワークです。
公式サイトではFastAPIについて、以下のように説明します。
FastAPI (公式ドキュメントより)
FastAPIは、Pythonの標準である型ヒントに基づいてPython3.6以降でAPI を構築するための、モダンで、高速(高パフォーマンス)な、Webフレームワークです。
FastAPI

FastAPIの特徴

公式サイトではFastAPIの特徴を以下のように説明しています。
  • 高速
  • 高速なコーディング
  • 少ないバグ
  • 直感的
  • 簡単
  • 短い
  • 堅牢性
  • Standards-based

ライブラリのインストール

FastAPIでは基本的に以下の2つのライブラリを使用します。
  • fastapi
  • uvicorn
これらをインストールします。 pip install fastapi pip install uvicorn[standard]
では、実際に動作をテストしましょう♪
「main.py」という名前のファイルを作成し、以下の内容を貼り付けてください。 from fastapi import FastAPI app = FastAPI() @app.get("/") def read_root(): return {"Hello": "World"} @app.get("/items/{item_id}") def read_item(item_id: int, q: str = None): return {"item_id": item_id, "q": q} 次に以下のコマンドで実際にWEBサーバを起動しましょう♪ uvicorn main:app --reload 実際にWEBサーバが正常に動作しているかは以下のURLにアクセスすることで確認できます。
  • http://127.0.0.1:8000/
  • http://127.0.0.1:8000/items/5?q=somequery
  • http://127.0.0.1:8000/docs
