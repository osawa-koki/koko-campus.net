ここでは良く用いる「Label」「Entry」「Button」の3つを説明します。

Label

Lebelウィジェットはテキストや画像を表示するために使用します。
Labelインスタンスを生成するには以下のように書きます。 label1 = tk.Label(親ウィジェット) 親ウィジェットにはrootを直接指定することもできますが、一般的にはウィジェットのコンテナとなるFrameにウィジェットをまとめて、そのFrameをrootに配置することが多いです。
ここではあくまでもLabelの動作を確認するだけなので、rootに直接配置することにします。

Labelでは以下の項目を設定可能です。
Label
text ラベルで表示する文字を指定します。
font 文字のスタイル・文字のサイズ・文字の太さを指定します。
タプル形式で「(フォントスタイル, フォントサイズ, フォント幅)」と指定します。
fg 文字色を指定します。
「foreground」と正式名称を使用することもできます。
bg 背景色を指定します。
「background」と正式名称を使用することもできます。
bd 枠線の幅を指定します。
「borderwidth」と正式名称を使用することもできます。
relief 枠線のスタイルを指定します。
Frameで説明した通りです。
textvaliable 可変文字列(StringVar)を使用する際に用います。
後ほど詳しく説明します。
label = tk.Label( root, text="koko-campus!", font=("consolas", 30, "bold"), fg="red", bg="yellow", bd=3, relief="solid" ) label.pack()

Entry

文字を入力するエリアを作成します。 entry = tk.Entry(親ウィジェット) 以下の項目が指定可能です。
Entry
text ラベルで表示する文字を指定します。
width サイズを指定します。
単位は半角文字一文字です。
font 文字のスタイル・文字のサイズ・文字の太さを指定します。
タプル形式で「(フォントスタイル, フォントサイズ, フォント幅)」と指定します。
fg 文字色を指定します。
「foreground」と正式名称を使用することもできます。
bg 背景色を指定します。
「background」と正式名称を使用することもできます。
relief 枠線のスタイルを指定します。
Frameで説明した通りです。
textvaliable 可変文字列(StringVar)を使用する際に用います。
後ほど詳しく説明します。
show 入力した文字を他の文字に変えて表示するかを指定します。
例えばパスワードの入力欄では「*」を指定して、「*****」と表示されるようにします。
command 入力された文字が変化したときに実行する関数を指定します。
詳しくはイベントハンドリングの授業で説明します。
entry = tk.Entry(root, font=("consolas", 16, "bold"), width=15, bd=3, relief="solid") entry.pack(pady=30)

Button

ユーザのクリックに応じて何らかの処理を実行します。
Button
width・height サイズを指定します。
単位はピクセルです。
font 文字のスタイル・文字のサイズ・文字の太さを指定します。
タプル形式で「(フォントスタイル, フォントサイズ, フォント幅)」と指定します。
fg 文字色を指定します。
「foreground」と正式名称を使用することもできます。
bg 背景色を指定します。
「background」と正式名称を使用することもできます。
relief 枠線のスタイルを指定します。
Frameで説明した通りです。
command ボタンがクリックされたときに実行する関数を指定します。
詳しくはイベントハンドリングの授業で説明します。
button = tk.Button(root,text="クリック!", font=("consolas", 16, "bold"), fg="red", bg="yellow") button.pack(pady=30)
