連想配列

pythonでは辞書型と呼びますが、他の言語に合わせてここでは連想配列と呼びます。
連想配列とは配列の進化バージョンです。
配列は配列内の要素を「0」から始まるインデックス番号で管理しますが、連想配列型では自分で名前を付けることができます。 連想配列は「キー: バリュー」とセットで書きます。
キーはバリューを特定するための識別子で重複は許されません。

連想配列の作成

連想配列は「{}(ブレース)」で表し、中身はキーとバリューを「:(コロン)」で区切ります。
複数個ある場合には「,(カンマ)」で区切って書きます。 asoc = { "いばらき": "なっとう", "さいたま": "せんべい", "とちぎ": "いちご", "にいがた": "おこめ", "あきた": "おこめ" }

連想配列の要素へのアクセス

連想配列の要素は「連想配列名[キー]」でアクセスします。 asoc = { "いばらき": "なっとう", "さいたま": "せんべい", "とちぎ": "いちご", "にいがた": "おこめ", "あきた": "おこめ" } print(asoc["さいたま"]) print(asoc["とちぎ"])

連想配列の走査

連想配列をforループで走査(ひとつずつ取り出す)方法は大きく以下の3通りあります。
  • keys
  • values
  • items

keys

配列のキーをひとつずつ取り出します。
以下のように実行します。 asoc = { "いばらき": "なっとう", "さいたま": "せんべい", "とちぎ": "いちご", "にいがた": "おこめ", "あきた": "おこめ" } for k in asoc.keys(): print(k) keysメソッドを用いなくてもそのまま連想配列をforループさせればキーが取り出せます。 asoc = { "いばらき": "なっとう", "さいたま": "せんべい", "とちぎ": "いちご", "にいがた": "おこめ", "あきた": "おこめ" } for k in asoc: print(k) キーを取得できれば「連想配列名[キー]」でバリューも取得できますが、後述するitemメソッドを用いた方が簡単です。

values

連想配列のバリューをforループで取り出します。 asoc = { "いばらき": "なっとう", "さいたま": "せんべい", "とちぎ": "いちご", "にいがた": "おこめ", "あきた": "おこめ" } for k in asoc.values(): print(k)

items

キーとバリューを同時にforループで取り出します。
以下のように記述します。 for キー用の配列, バリュー用の配列 in 連想配列: # 反復処理... 簡単にキーとバリューを取得できます。
他の言語でバリューを取得するには「連想配列名[キー]」とわざわざ書かなければいけないんですけをPythonではキーもバリューもともにfor文でループさせることができます。 asoc = { "いばらき": "なっとう", "さいたま": "せんべい", "とちぎ": "いちご", "にいがた": "おこめ", "あきた": "おこめ" } for k, v in asoc.items() : print(k + " と言えば " + v)

要素の追加

以下のコードで実行します。 連想配列[追加するキー] = 追加するバリュー 追加するキーが既に存在する場合にはバリューのデータを上書きします。 asoc = { "いばらき": "なっとう", "さいたま": "せんべい", "とちぎ": "いちご", "にいがた": "おこめ", "あきた": "おこめ" } asoc["おきなわ"] = "ゴーヤ" # 「おきなわ」キー・「ゴーヤ」バリューを追加 asoc["あきた"] = "いぶりがっこ" # 「あきた」キーのバリューを「いぶりがっこ」に更新 for k, v in asoc.items(): print(k + " と言えば " + v)

要素の削除

配列と同様に「del」文で連想配列を削除できます。 asoc = { "いばらき": "なっとう", "さいたま": "せんべい", "とちぎ": "いちご", "にいがた": "おこめ", "あきた": "おこめ" } del asoc["にいがた"] for k, v in asoc.items(): print(k + " と言えば " + v)
