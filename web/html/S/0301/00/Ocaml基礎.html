データ型については後ほど説明しますが、整数・小数・文字列などのデータの種類を表すのもだと思ってください。 let () = print_bytes (Bytes.of_string "Iam bytes!"); print_char 'A'; print_endline "Hello Ocaml"; print_newline (); print_int 10; print_float 10.25;

変数

変数とはプログラムの実行に際して必要となる一時的なデータを保存するための箱です。
変数は以下のように宣言します。 let 変数 = 値; 注意するべき点は変数は不可変(イミュータブル)である点です。
関数型では一度変数に値を代入したらこれを変更することができません。

もはや変数とも呼ばない気がしますね、、、
したがって以下のコードはエラーとなります。 {- コンパイルエラー -} x = "ocaml" x = "ocaml" main = do putStr x {- ↓ コンソール ↓ Multiple declarations of ‘x’ -} ocamlでは変数の不可変という性質から変数に値を代入すると言う代わりに、変数に値を拘束すると言うことが多いです。
値を変更できる真の変数も使用することができますが、ocamlを使用する上で特別な理由がない限りは使用するべきではありません。
参考までに紹介します。 import Data.IORef -- 「Data.IORef」モジュールを使用 main = do v <- newIORef "ocaml" -- vに「ocaml」を代入 show1 <- readIORef v -- vの値をshow1に拘束 putStrLn show1 -- show1の値を出力 writeIORef v ("ocaml") -- vの値を更新 show2 <- readIORef v -- vの値をshow2に拘束 putStrLn show2 -- show2の値を出力
