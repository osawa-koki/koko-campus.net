データベース

プログラムがデータを永続化されるためには、以下の2通りが考えられます。
  • ファイルへの保存
  • データベースへの保存

データベースも結局はファイルから構成されていますが、、、

ファイルでの保存は非常に操作がしにくく、かつコードの可読性の低下につながるという問題があります。
また、データ構造も複雑となり、アルゴリズムに対しても悪い影響を与えることが多いです。
データベースはデータの管理(追加・削除・更新・参照)のプロフェッショナルですので、データの管理は自分でファイル操作を行うのではなく、データベースに任せたほうが楽チンです。

このような理由から、データを保存するには一般的にデータベースを使用することが多いです。
ということで、C#でデータベースを操作する方法を学習しましょう♪

SQL Server

実際に我々がデータベースを直接操作するのではなく、DBMSと呼ばれるプログラムに命令を出して、データベースを操作してもらいます。 DBMSに対する命令はSQLと呼ばれ、実際にデータベースを操作するにはこのSQLを書く作業が大半となります。

DBMSにはいくつかの種類があります。
代表的なものには以下のDBMSがあげられます。
  • MySQL
  • Oracle DB
  • Mongo DB
  • SQLite
  • SQL Server
ここでは、C#と開発元であるマイクロソフトが開発していて、相性がいい「SQL Server」を使用します。

以下のリンクから「SQL Server」をインストールして下さい。 SQL Server 「Express」バージョンをダウンロードして下さい。

ダウンロードが完了したらそのまま起動し、インストールを完了させます。
インストール時にいくつかの設定をする場面がありますが、全部デフォルトでOK!です。

次にデータベースを簡単に操作するためのツールであるSSMSをインストールしましょう♪ SSMS こちらも設定は全てデフォルトでOK!です。

インストールが完了したら、SSMSを起動して下さい。
ログイン画面が現れたら、そのままログインしてください。
「PC名\SQLEXPRESS」でログイン可能です。
認証方式は「Windows Authentication」を選択して下さい。

ログインに成功したら、左上の「New Query」を押して、SQLエディタを表示し、そこにSQLを書いて「Execute」ボタンで実行します。
